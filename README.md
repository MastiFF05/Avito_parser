# Avito Cars Parser
Парсер автомобилей с Avito.ru (Москва). Собирает название, цену, параметры и ссылки на объявления с пагинацией.

## Функционал
- Пагинация
- Обработка ошибок сети
- Задержки между запросами
- Сохранение в CSV с русскими символами
- requests.Session() для скорости

## Пример результата

| Название          | Цена        | Дополнительные параметры  | Ссылка   |
|-------------------|-------------|---------------------------|----------|
| Toyota Camry 2020 | 2 500 000 ₽ | 2020 • 50 000 км • Бензин | [ссылка] |

## Быстрый старт

# Клонируем
git clone https://github.com/MastiFF05/Avito_parser.git
cd avito_parser

# Устанавливаем зависимости
pip install -r requirements.txt

# Запускаем
python main.py
Результат: avito_cars.csv готов к анализу!

Настройки в коде
max_pages=2  # Количество страниц для парсинга
DELAY=2      # Задержка между запросами (сек)

Используемые библиотеки
requests    # HTTP клиент для запроса к сайту
BeautifulSoup # HTML парсинг
csv         # Экспорт данных с csv-файл
time        # Для задержек между запросами
